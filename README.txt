# Prueba E2E con Cypress y POM

Este proyecto realiza una prueba funcional automatizada (Prueba E2E) del flujo de compra en la página "https://www.Demoblaze.com".

## Flujo de Prueba

1. Agregar dos productos al carrito de compras.
2. Visualizar el carrito de compras.
3. Completar el formulario de compra.
4. Finalizar la compra.

## Requisitos

- Node.js
- Cypress

## Cómo Ejecutar la Prueba

1. Clona este repositorio.
2. Instala las dependencias con `npm install`.
3. Ejecuta las pruebas con `npx cypress open` y selecciona `purchaseFlow.spec.js` para ejecutar la prueba.
ó puedes ejecutar el siguiente script directamente en la terminal: "npm run chrome"

## Tecnologías y Herramientas

- Cypress: Para pruebas E2E.
- Modelo de Objetos de Página (POM)
